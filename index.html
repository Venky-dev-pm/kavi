Frequently asked questions

Is KAVI really 100% local?
Yes. All AI processing (voice recognition, language model, text-to-speech) runs on your CPU. The only network traffic is optional OAuth for Google/Outlook calendar sync — and that only sends calendar data, never your voice, tasks, or meetings.
What are the system requirements?
Any modern laptop with 4GB+ RAM. No GPU required — KAVI uses 4-bit quantized models optimized for CPU inference. Total memory usage stays under 1.5GB with all features active.
Does it work on Windows, Mac, and Linux?
Yes. KAVI is built with Tauri v2, which compiles native binaries for all three platforms. System audio capture works via WASAPI (Windows), ScreenCaptureKit (macOS), and PulseAudio (Linux).
Is KAVI open source?
The core engine will be open source so anyone can audit the code and verify that your data stays local. We believe transparency is essential for a privacy-first product.
When will KAVI be available?
We're targeting a beta launch in Q2 2026. Waitlist members get first access. Join above to be among the first to try KAVI.
How does voice recognition work without internet?
KAVI uses Whisper tiny.en, a speech recognition model that runs entirely on CPU. It's bundled with the app — no internet connection needed for any AI features. The model is only 75MB and processes speech in real-time.
